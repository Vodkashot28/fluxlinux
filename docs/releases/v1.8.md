# FluxLinux v1.8.0 Release Notes

### 1. Uninstall Option for Feature Packages (T1)
Users can now cleanly remove installed feature packages without re-running scripts or wiping the distro:
- **Uninstall button** on each component card in Distro Settings
- **13 component scripts** updated with `uninstall` branch that removes packages, cleanup wrappers, .desktop files, venvs, and reverts shell config changes
- **Confirmation dialog** before uninstall with size estimate
- **State management**: component marked as not-installed only on successful completion
- **Hidden for**: mandatory components, coming-soon components, and XFCE4 Desktop (uses base-install script)

### 2. Foreground Service for Install Server (T3)
Android's background process killer no longer breaks the install bridge:
- **InstallServerService**: new foreground service hosts the local HTTP install server
- **Persistent notification** keeps the process alive when the app is backgrounded
- **5-minute idle timer** for base-install path (matches previous behavior)
- **Auto-stop** when install queue drains or user cancels
- **Port discovery**: activity awaits on a StateFlow for the bound port

### 3. Termux Version Warning Banner (T4)
New users with outdated Play Store Termux now see a prominent warning:
- **Top-of-screen banner** when installed Termux < v0.118.3
- **Inline warning** on the Termux prerequisite card with download button
- **Continue button** disabled until correct version is installed
- **SHA256 verification** on downloaded APKs with corrupt APK redownload option

### 4. Monochrome Themed Icon (T6)
- **Single-color vector** monochrome layer added to adaptive icon
- **Material You / Dynamic Color** support on Android 13+
- **Themed icons** now work on supported launchers

### 5. Termux Native Desktop Support (T9, T10, T11)
Termux Native is now fully enabled (no longer "coming soon"):
- **XFCE4 Desktop**: native XFCE4 via Termux:X11 — no container overhead
- **KDE Plasma Desktop**: full KDE running natively in Termux
- **Hardware Acceleration**: VirGL & Turnip GPU drivers for native acceleration
- **Customization**: FluxLinux theme, wallpapers, and fonts for both DEs
- **KDE launch fixes**: proper Turnip/Software renderer detection, dark splash screen in dark theme

### 6. Script Restructure (Refactoring)
Scripts reorganized into a per-distro hierarchy for clarity:
- `common/` → `debian/common/setup/`, `debian/common/addon/`, etc.
- `termux/` — native Termux scripts
- `arch/` — Arch-based distro scripts
- **setup/start/stop/addon** sub-folders within each distro

### 7. Build & Quality Improvements
- **Concurrent install prevention** (T2): verified already implemented — queue-based install manager
- **Exit code propagation**: all install paths now capture script exit codes and fire success/error callbacks
- **Deep link callback fix**: strict task ID matching prevents stale callback attribution
- **Dark mode forced**: theme picker removed, app now permanently dark
- **Discord card + tutorial docs** added to Settings screen
- **Version string updated**: Settings now shows v1.8.0 • Jun 17, 2026

### Migration Notes
No breaking changes. No migration required.
