# Setting up FluxLinux

Follow these instructions to download and install FluxLinux on your Android device.

## Step 1: Download the Application

You can obtain FluxLinux through one of the following official channels:

### 1. GitHub Releases (Free)
Download the latest pre-built APK directly from the GitHub repository.
* **Link:** [GitHub Releases](https://github.com/abhay-byte/fluxlinux/releases)
* **Screenshot:**
  ![GitHub Release Download](img/release-download.png)

### 2. F-Droid (Free & Open Source)
Install via the F-Droid client or download the APK from the F-Droid website.
* **Link:** [F-Droid Listing](https://f-droid.org/packages/com.ivarna.fluxlinux)
* **Screenshot:**
  ![F-Droid Download](img/fdroid-download.png)

### 3. Google Play Store (Paid/Support Developer)
Show your support by purchasing the app on the Google Play Store.
* **Link:** [Google Play Store](https://play.google.com/store/apps/details?id=com.ivarna.fluxlinux)
* **Screenshot:**
  ![Google Play Store Download](img/playstore-download.png)
